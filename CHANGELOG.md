## 2026-05-01 — финальный спринт перед тестом заказчика

### PR #31 — `feat: 6 категорий + 100Q replay тест + CHANGELOG`
- **A.** RAG листовки расширен с 6 до **12 категорийных чанков**: добавлены рыба/морепродукты, бакалея, замороженное, снеки, бытхимия/гигиена, корма.
- **B.** `scripts/replay_real_queries.py` — прогон 100 случайных запросов из реальных логов прода с замером latency, гистограммой по категориям, top-10 самых медленных.
- **D.** Этот файл `CHANGELOG.md`.

### PR #30 — `feat(rag+pii+welcome): категорийные чанки + Краш Прайс whitelist + Welcome`
- **3 микро-фикса** по результатам Telethon-теста 18Q:
- **PII whitelist** для «Краш Прайс» / «Lucky Goods» — natasha NER принимала за ФИО.
- **6 категорийных RAG-чанков** в листовке (молочка/мясо/хлеб/овощи/напитки/кондитерка) — на запросы «акции на молочку» бот теперь возвращает реально молочные товары, не шоколад.
- **Welcome message** обновлён на актуальные примеры.

### PR #29 — `fix(prompt): §11b explicit правило для блока «Цена вниз с Еплюс»`
- Промпт §11b: на «цены с картой Еплюс» бот теперь обязан выдавать **товары** из chunk'а «Цена вниз с Еплюс», а не сводить ответ к «копите бонусы».
- Закрыло ⚠️ #6 из `final_regress_after_PR28.md`.

### PR #28 — `fix(cache+filter): TTL response_cache 60→15 мин + content_filter политики`
- **TTL 3600 → 900** секунд: после деплоя свежие правки доходят до пользователя в 4× быстрее (было: бот возвращал старые ответы из кэша до 1 часа после фикса).
- **content_filter:** добавлены `парти лучше`, `какая парти`, `государств` — закрывает 2 false-negative сценария от заказчика 28.04.
- Регрессия: `партия товара`, `гос.праздник` продолжают проходить.

### PR #27 — `feat(autolavki+evroshok+validator)`
- **A6:** `load_avtolavki_chunks()` — 1 агрегатный chunk из 55 автолавок по городам-базам с явным «маршруты не публикуются → горячая линия».
- **B7/G4:** canonical `edostavka_promo_no_info` теперь даёт **осмысленный** ответ про Е-доставку как отдельный сервис (раньше: «нет информации»).
- **Усиленный validator** в `agent_tester_telethon.py`: 3 семантические проверки (пустой список после preamble, refusal на категорийный запрос с листовкой, евро-шок без редиректа).

### PR #26 — `fix(grounding): fuzzy address match + time auto-fix`
- **Fuzzy address match:** verifier перестал резать легитимные адреса при перефразировке LLM (`«ул. Шаранговича В.Ф., 48»` ↔ `«ул. Шаранговича, 48»`). Корень улицы + номер дома в радиусе 120 символов.
- **Hours safe-fact:** «8:00–23:00» добавлено в `SAFE_FACTS["hours"]` — каноническое время гипермаркетов из SYSTEM_PROMPT больше не флажится.
- **Time auto-fix:** удаление целого предложения вместо `replace` (раньше получалось «работы — с уточните на evroopt.by/shops»).
- +7 unit-тестов, всего 27/27 PASS.

### PR #25 — `fix(pii+rag): «Родныя тавары» не блокировать как ФИО + ключевые слова листовки`
- **PII фильтр:** добавлены `_FIO_STOPWORDS` для брендов/акций («Родныя тавары», «СВАЁ», «Удача в придачу», «Грильфест»). NER natasha больше не принимает их за ФИО.
- **RAG keywords:** в `listovka_summary` добавлены категорийные ключи (молочка/мясо/хлеб/овощи/...). Бот цепляет snapshot листовки на «акции на молочку» вместо отказа.
- +10 параметризованных PII-кейсов, 20/20 PASS.

### PR #24 — `feat(listovka): snapshot текущей листовки 27.04–10.05.2026 в RAG`
- `scripts/parse_listovka_xlsx.py` — парсер xlsx-снимка от заказчика → JSON.
- `scripts/reindex_v2.py::load_listovka_current()` — **9 chunks** в RAG: 1 summary + 6 per-promo + СВАЁ + Родныя.
- 300 товаров с ценами и периодом действия загружены.
- Закрыло категорийные запросы про акции (молочка/мясо/...).
- +4 smoke-теста.
